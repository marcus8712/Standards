Text Alignment

A header can have its text aligned to a side. Left or right aligned text is sticked with the margin. When a block of text is justified, each line fills the entire space from left to right.

Left

Right

Center

This should take up the full width even if only one line

Usage : insert a class .aligned before the class .header.

    Left
</h4>

    Right
</h4>

    Center
</h4>
