Input

Input fields are used to gather short information from a user. An input field can be used to gather numerical data or text responses.

Usage : add classes ui, input to the <div> element. 


<div class="ui input">
  <input type="text" placeholder="Search...">
</div>

States

Focus

An input field can show a user is currently interacting with it.

Usage : insert a class .focus before the class .input. 


<div class="ui focus input">
  <input type="text" placeholder="Search...">
</div>

Loading

Usage :
- Insert classes .icon, .loading before the class .input.
- By default, loading icon is place at the right of input. Add a class .left before the classes .icon, .loading to align left. 


<div class="ui icon loading input">
  <input type="text" placeholder="Search...">
  <i class="search icon"></i>
</div>

<div class="ui left icon loading input">
  <input type="text" placeholder="Search...">
  <i class="search icon"></i>
</div>

Disabled

Usage : insert a class .disabled before the class .input. 


<div class="ui disabled input">
  <input type="text" placeholder="Search...">
</div>

Error

An input field can show the data contains errors.

Usage : insert a class .error before the class .input. 


<div class="ui error input">
  <input type="text" placeholder="Search...">
</div>

Variations

Icon

An input is formatted with an icon.

Usage : insert a class .icon before the class .input. Include a class .left for left aligned icon. 


<div class="ui icon input">
  <input type="text" placeholder="Search...">
  <i class="search icon"></i>
</div>

<!-- Left --> 
<div class="ui left icon input">
  <input type="text" placeholder="Search users...">
  <i class="user icon"></i>
</div>

Transparent

A input is formatted with non-background.





Usage : insert a class .transparent before the class .input. 


<div class="ui transparent input">
  <input type="text" placeholder="Search...">
</div>

<!-- Right Icon --> 
<div class="ui icon transparent input">
  <input type="text" placeholder="Search...">
  <i class="search icon"></i>
</div>

<!-- Left Icon -->                        
<div class="ui left icon transparent input">
  <input type="text" placeholder="Search...">
  <i class="search icon"></i>
</div>

Inverted

An input is formatted to appear on dark backgrounds

Usage : insert a class .inverted before the class .input. 


<div class="ui inverted input">
    <input type="text" placeholder="Search...">
</div>

Labeled

An input is formatted with a text label, a linked label or a dropdown label.

Text Label
http://
kg

Usage : insert a class .labeled before the class .input. Label is default on the left side of input, add a class .right to put it at the right side. 


<div class="ui labeled input">
  <div class="ui label">http://</div>
  <input type="text" placeholder="mysite.com">
</div>

<!-- Right Label --> 
<div class="ui right labeled input">
  <input type="text" placeholder="Enter weight..">
  <div class="ui basic label">kg</div>
</div>
Linked Label
Add Tag 

<div class="ui right labeled left icon input">
  <i class="tags icon"></i>
  <input type="text" placeholder="Enter tags">
  <a class="ui tag label">
    Add Tag
  </a>
</div>
Dropdown Label
.com
.com
.net
.org

<div class="ui input right labeled">
  <input type="text" placeholder="Find domain">
  <div class="ui dropdown label">
    <div class="text">.com</div>
    <i class="dropdown icon"></i>
    <div class="menu">
      <div class="item">.com</div>
      <div class="item">.net</div>
      <div class="item">.org</div>
    </div>
  </div>
</div>

Action

An input is formatted to alert the user to an action they may perform

Text and icon action

Usage : insert a class .action before the class .input. 


<div class="ui action input">
  <input type="text" placeholder="Enter number">
  <button class="ui button">Search</button>
</div> 

<div class="ui action input">
  <input type="text" placeholder="Search...">
  <button class="ui button icon"><i class="search icon"></i></button>
</div>
Primary Action

Usage : insert a class .primary before the class .button. 

                           
<div class="ui action input">
  <input type="text" placeholder="Enter number">
  <button class="ui primary button">Search</button>
</div>
Left side action
                           
<div class="ui left action input">
  <button class="ui button info labeled icon">
    <i class="cart icon"></i> Checkout
  </button>
  <input type="text" value="$52.03">
</div>
Dropdown action
This Page
This Organization
Entire Site
                           
 <div class="ui input right action left icon">
  <i class="search icon"></i>
  <input type="text" placeholder="Search">
  <div class="ui dropdown button">
    <div class="text">This Page</div>
    <i class="dropdown icon"></i>
    <div class="menu">
      <div class="item">This Organization</div>
      <div class="item">Entire Site</div>
    </div>
  </div>
</div>
