Component

List

Lists are used to group items together in order to view and access easily. By default, a list is displayed without list-style-type property.

Apples
Pears
Oranges

Usage : add classes .ui, .list to the <div> element. 


<div class="ui list">
  <div class="item">...</div>
</div>

Contents

Header

A list item can contain a header.

New York City
A lovely city
Chicago
Also quite a lovely city
Los Angeles
Sometimes can be a lovely city
San Francisco
What a lovely city

Usage : add a <div> element with a class .header into the item. 


<div class="ui list">
  <div class="item">
    <div class="header">New York City</div>
    A lovely city
  </div>
</div>

Description

A list item can contain a description.

New York City
A lovely city
Chicago
Also quite a lovely city
Los Angeles
Sometimes can be a lovely city
San Francisco
What a lovely city

Usage : insert a <div> element with a class .description after the header. 


<div class="ui list">
  <div class="item">
    <a class="header">New York City</a>
    <!-- Description -->
    <div class="description">A lovely city</div>
  </div>
</div>

Icon

A list item can contain an icon. It is splitted into left block and content block.

New York City
A lovely city
Chicago
Also quite a lovely city
What a lovely city

Usage :
- Wrap the header and text in a content block.
- Insert a <i> element with a class .icon before the content block.
- Text placed next to icon is the inline text. 


<div class="ui list">
    <a class="item">
        <i class="location icon"></i>
        <div class="content">
            <div class="header">New York City</div>
            A lovely city
        </div>
    </a>
    <!-- Inline Text -->
    <div class="item">
        <i class="help icon"></i>
        Inline Text
    </div>
</div>

Image

A list item can contain an image that is put into left block.

Rachel
Last seen watching Arrested Development just now.
Lindsay
Last seen watching Bob's Burgers 10 hours ago.
Matthew
Last seen watching The Godfather Part 2 yesterday.

Usage : replace the icon element with an image element. 


<div class="ui list">
    <div class="item">
        <img class="ui avatar image" src="rachel.png">
        <div class="content">
            <a class="header">Rachel</a>
            <div class="description"></div>
        </div>
    </div>
</div>

Types

Unordered List

A unordered list is used to group a set of related items, in no particular order.

Orange
Green
Blue

Usage : insert a class .bulleted before the class .list.


<div class="ui bulleted list">
  <div class="item">...</div>
</div>

Ordered List

An ordered list is used to group a set of related items, in a specific order. It works with numbers instead of using a dot as the default list item marker.

View items
Select an item
Export the item

Usage : insert a class .ordered before the class .list.


<div class="ui ordered list">
  <div class="item">...</div>
</div>

Link

A list item is formed as navigation links.

Home
About Jobs Team

Usage : insert a class .link before the class .list.


<div class="ui link list">
  <div class="active item">...</div>
  <a class="item">...</a>
</div>
