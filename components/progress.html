Progress

Progress bars is used to show that assets are loading, in progress, or that there is action taking place regarding elements on the page.

Initializing a progress

Call an identifier via a single line of JavaScript to initiate a progress. 


<script>
$('.ui.progress').progress();
</script>

Content

A progress can contain a label and a bar which displays a text value indicating current progress.

Simple Progress

Usage :
- Add a <div> with classes .ui, .progress.
- Next, inside the above <div>, add an empty <div> with a class .bar. 


<div class="ui progress">
    <div class="bar"></div>
</div>
Progress with a text value

Usage : add a <div> with a class .progress into the bar.


<div class="ui progress">
    <div class="bar">
        <div class="progress"></div>
    </div>
</div>
Progress with a label
Uploading Files

Usage : insert a <div> with a class .label after the bar.


<div class="ui progress">
    <div class="bar">
        <div class="progress"></div>
    </div>
    <div class="label">Uploading Files</div>
</div>

States

Active

A progress bar can show activity.

Uploading Files

Usage : insert a class .active before the class .progress.


<div class="ui active progress">
    <div class="bar"></div>
    <div class="label">Uploading Files</div>
</div>

Disabled

A progress bar can show an error state.

Usage : insert a class .error before the class .progress.


<div class="ui error progress">
    <div class="bar">
        <div class="progress"></div>
    </div>
    <div class="label">...</div>
</div>
