Image

Image is used to provide a visual representation of context. It should be specified the width and height. If width and height are not specified, the page will flicker while the image loads.

Usage : An image is defined with the img tag which contains attributes only, and does not have a closing tag. Or it is created with a <div> element and classes .ui, .image. 


<img class="ui image" src="image.jpg" alt="image">  
                    
<div class="ui image">
  <img src="image.jpg" alt="image">
</div>

State

Disabled

Usage : insert a class .disabled before the class .image.


<img class="ui disabled image" src="image.jpg" alt="image">

Type

Link

An image can be formatted to link to other content.

Username

Usage : insert a class .avatar before the class .image.


<img class="ui avatar image" src="image.png">
<span>Username</span>
