Item

An item

An item is used with its full-width block or divided up into 2 blocks which left block containing image or icon, right block containing content.

Jenny Hess

Jenny is a student studying Media Management at the New School.

Usage : add a class .item to the <div> element. 


<div class="item">
    Inside item
</div>

Items

An item never stands alone. Instead, it is always worked with a list, a group of items, etc. You can view the list at here.

Jenny Hess

Jenny is a student studying Media Management at the New School.

Stevie Feliciano

Stevie Feliciano is a library scientist living in New York City. She likes to spend her time reading, running, and writing.

Usage : add classes .ui, .item to the <div> element. Add item divs into it.


<div class="ui items">
    <div class="item">
        Item 1
    </div>
    <div class="item">
        Item 2
    </div>
</div>

Contents

Full-Width Block

An item is used as a wrapper containing any components, like an image, a header, a table, etc.

Usage : add any components (e.g. image) into item block. 


<div class="item">
    <div class="ui image"><img src="image.jpg"></div>
</div>

Left block and Content block

The content of an item is quite broad, including metadata, paragraphs, lists, performatted text, forms, tables and figures. An item without left block is displayed as the full-width block.

Left block contains an image or an icon.

Header
Veronika Ossi

Usage : add header into content block.


<div class="item">
    <div class="ui image"><img src="image.jpg"></div>
    <div class="content">
        <!-- Header -->
        <div class="header">Header</div>
    </div>
</div>
Metadata
Arrowhead Valley Camp
$1200 1 Month

Usage : add meta into content block.


<div class="item">
    <div class="ui image"><img src="image.jpg"></div>
    <div class="content">
        <!-- Metadata -->
        <div class="meta">
            <span class="price">$1200</span>
            <span class="stay">1 Month</span>
        </div>
    </div>
</div>
Description
Cute Dog

Cute dogs come in a variety of shapes and sizes. Some cute dogs are cute for their adorable faces, others for their tiny stature, and even others for their massive size.

Many people also have their own barometers for what makes a cute dog.

Usage : add description into content block.


<div class="item">
    <div class="ui image"><img src="image.jpg"></div>
    <div class="content">
        <div class="header">Cute Dog</div>
        <!-- Description -->
        <div class="description">
            <p></p>
        </div>
    </div>
</div>
Extra Content

A item can contain extra content meant to be formatted separately from the main content.

Cute Dog
121 Votes

Usage : add extra into content block.


<div class="item">
    <div class="content">
        <div class="header">Cute Dog</div>
        <div class="description">
            <p></p>
        </div>
        <!-- Extra -->
        <div class="extra">
            <i class="primary check icon"></i>
            121 Votes
        </div>
    </div>
</div>

Variation

Floated Content

Any content element can be floated left or right.

Content
Action

Usage : add classes .left, .floated or .right, .floated to the class content.


<div class="item">
    <div class="ui image"><img src="image.jpg"></div>
    <div class="content">
        <div class="header">Content A</div>
        <div class="description"><p></p></div>
        <div class="extra">
            <div class="ui right floated button">
                Action
            </div>
        </div>
    </div>
</div>

Groups

Vertical Alignment

The content of items can specify its vertical alignment.

Top Aligned
Middle Aligned
Bottom Aligned

Usage : insert a class .aligned before the class content.


<div class="ui items">
    <div class="item">
        <div class="ui image"><img src="../source/img/image.jpg"></div>
        <div class="top aligned content">
            <div class="header">Top Aligned</div>
        </div>
    </div>
    <div class="item">
        <div class="ui image"><img src="../source/img/image.jpg"></div>
        <div class="middle aligned content">
            <div class="header">Middle Aligned</div>
        </div>
    </div>
    <div class="item">
        <div class="ui image"><img src="../source/img/image.jpg"></div>
        <div class="bottom aligned content">
            <div class="header">Bottom Aligned</div>
        </div>
    </div>   
</div>
