Definition Types Variations

Page Headers

The header represents a container for introductory content or a set of navigational links.

First header

Second header

Third header

Fourth header

Fifth header

Usage : add classes .ui, .header to the <heading> element. 


<h1 class="ui header">First header</h1>
<h2 class="ui header">Second header</h2>
<h3 class="ui header">Third header</h3>
<h4 class="ui header">Fourth header</h4>
<h5 class="ui header">Fifth header</h5>

Content Header

A header can contain an image, an icon and a subheader.

An Image

Hoiio Pte Ltd

Usage : add an <image> element into header block. 


<h2 class="ui header">
  <img class="ui image" src="image.jpg">
  <div class="content">
    Hoiio Pte Ltd
  </div>
</h2>
An Icon

Page Title

Usage : insert an <i> element with a class .icon before the content block.
A SubHeader

Account Settings
Manage your account settings and set e-mail preferences.

Usage : insert an <div> element with classes .sub, .header after the heading text. 


<h2 class="ui header">
  Account Settings
  <div class="sub header">Manage your account settings and set e-mail preferences.</div>
</h2>

Sub Headers

Headers is formed to label smaller or de-emphasized content.

Price

$10.99

Usage : insert a class .sub before the class .header. 


<h2 class="ui sub header">Sub header</h2>

Attached

Top Attached
Attached

Floating

A header can sit to the left or right of other content.

Go Forward

Go Back

Usage : insert classes .left, .floated or .right, .floated before the class .header element. 


<h3 class="ui right floated header">
    Go Forward
  </h3>
  <h3 class="ui left floated header">
    Go Back
  </h3>

Text Alignment

A header can have its text aligned to a side. Left or right aligned text is sticked with the margin. When a block of text is justified, each line fills the entire space from left to right.

Left

Right

Center

This should take up the full width even if only one line

Usage : insert a class .aligned before the class .header. 


<h4 class="ui left aligned header">
    Left
</h4>
<h4 class="ui right aligned header ">
    Right
</h4>
<h4 class="ui center aligned header">
    Center
</h4>
<h4 class="ui justified header">
    This should take up the full width even if only one line
</h4>
