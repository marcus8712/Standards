Modal

A modal is a child window that is layered over its parent window. Typically, the purpose is to display content from a separate source that can have some interaction without leaving the parent window. Child windows can provide information, interaction, or more.

Initializing a modal

Toggle the modal's hidden content via JavaScript : call an identifier of modal with a single line of JavaScript. 


<script>
$('.demo.modal').modal('show')
</script>

Attach events to another element, like button. 


<script>
// Clicking the demo button will show the demo modal.
$('.demo.modal')
  .modal('attach events', '.demo.button')
;
</script>

A static modal window

A modal contains a header, a close icon, content and actions. Close actions are applied by default to all button actions, in addition an onApprove or onDeny events will fire if the elements match either selector. 

approve  : '.approve, .ok'
deny     : '.deny, .cancel'

Usage :
- Add classes .ui, .modal to the <div> element.
- Include header, content and actions .


<div class="ui demo modal">
    <i class="close icon"></i>
    <div class="header">
        This is the modal title
    </div>
    <div class="content">
        <p>Add some text here.</p>
    </div>
    <div class="actions">
        <div class="ui deny button">
            Cancel
        </div>
        <div class="ui approve primary button">
            Submit
        </div>
    </div>
</div>

            <div class="ui header">Default Profile Image</div>
            <p></p>

    </div>

</div>
