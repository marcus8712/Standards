Component

Segment

A segment is used to create a grouping of related content. It covers content by a border.

Usage : add classes .ui, .segment to the <div> element.


<div class="ui segment">
    <p></p>
</div>

States

Disabled

Disable the whole segment.

Usage : insert a class .disabled before the class .segment.


<div class="ui disabled segment">
  <p></p>
</div>

Loading

A segment may show its content is being loaded.

Usage : insert a class .loading before the class .segment.


<div class="ui loading segment">
  <p></p>
</div>

Types

Raised

A segment is formed to raise above the page.

Usage : insert a class .raised before the class .segment.


<div class="ui raised segment">
    <p></p>
</div>

Vertical Segment

A vertical segment is formatted content to be aligned as part of a vertical group.

Usage : insert a class .vertical before the class .segment.


<div class="ui vertical segment">
    <p></p>
</div>
<div class="ui vertical segment">
    <p></p>
</div>
<div class="ui vertical segment">
    <p></p>
</div>

Variations

Attached

A segment is attached to other content on a page and sometimes used with other attached variations like attached header or attached messages.

This segment is on top

This segment is attached on both sides

This segment is on bottom

Usage : insert classes .top .attached, .attached or .bottom .attached before the class segment.


<div class="ui top attached segment">
  <p>This segment is on top</p>
</div>
<div class="ui attached segment">
  <p>This segment is attached on both sides</p>
</div>
<div class="ui bottom attached segment">
  <p>This segment is on bottom</p>
</div>

Padded

A segment can increase its padding.

Usage : insert classes .padded (padding: 2em) or .very, ipadded (padding: 3em) before the class .segment.


<div class="ui padded segment">
    <p></p>
</div>

<div class="ui very padded segment">
    <p></p>
</div>

Compact

A segement is took width according to its content automatically.

Usage : insert a class .compact before the class .segment.


<div class="ui compact segment">
    <p></p>
</div>

Floated

A segment can appear to the left or right of other content.

This segment will appear to the right

This segment will appear to the left

Usage : insert classes .left, .floated or .right, .floated before the class .segment.


<div class="ui right floated segment">
    <p></p>
</div>

<div class="ui left floated segment">
    <p></p>
</div>

Text Alignment

A segment can have its text aligned to a side. If it is wrapped by a grid, please check alignment between blocks at Grid Alignment.

Left
Center
Right

Usage : insert classes .left, .aligned or .center, .aligned or .right, .aligned before the class .segment.


<div class="ui left aligned segment"> Left </div>
<div class="ui center aligned segment"> Center </div>
<div class="ui right aligned segment"> Right </div>

Clearing

A segment can clear floated content.

Floated

Usage : insert a class .clearing before the class .segment. 


<div class="ui clearing segment">
  <div class="ui right floated button">Floated</div>
</div>

Basic

A basic segment has no special formatting.

Pellentesque habitant morbi tristique senectus et netus et malesuada fames ac turpis egestas. Vestibulum tortor quam, feugiat vitae, ultricies eget, tempor sit amet, ante. Donec eu libero sit amet quam egestas semper. Aenean ultricies mi vitae est. Mauris placerat eleifend leo.

Usage : insert a class .basic before the class .segment. 


<div class="ui basic segment">
  <p>...</p>
</div>

Segments

Segments

A group of segments can be formatted to appear together.

Top

Middle

Bottom

Primary Content

Secondary Content

Compact Content

Compact Content

Usage : a segments div contains all segement divs.


<div class="ui segments">
  <div class="ui segment">Top</div>
  <div class="ui segment">Middle</div>
  <div class="ui segment">Bottom</div>
</div>

<!-- Secondary -->
<div class="ui segments">
  <div class="ui segment">Primary Content</div>
  <div class="ui secondary segment">Secondary Content</div>
</div>

<!-- Compact -->
<div class="ui compact segments">
  <div class="ui segment">Compact Content</div>
  <div class="ui segment">Compact Content</div>
</div>

Horizontal Segments

A segment group can appear horizontally

Left

Center

Right

Usage : prefix horizontal to segments.


<div class="ui horizontal segments">
  <div class="ui segment">
    <p>Left</p>
  </div>
  <div class="ui segment">
    <p>Center</p>
  </div>
  <div class="ui segment">
    <p>Right</p>
  </div>
</div>

Top

Nested Bottom

Left
