Component

Label

A label provides additional information about something that is often used associate to a form.

This is a label
23
Tag

Usage : add classes .ui, .label to the <div> element. 


<div class="ui label">
  This is a label
</div>

Contents

Detail

A label can contain a detail, an icon or an image

Dogs
214
Mail
Elliot

Usage :
- Add a <div> element with a class .detail into the class label.
- Add an <i> element with a class .icon into the class label.
- Add an <image> element to the class label. 


<div class="ui label">
  Dogs
  <!-- Detail -->
  <div class="detail">214</div>
</div>

<div class="ui label">
  <i class="mail icon"></i>
  Mail
</div>

<div class="ui label">
  <img class="ui avatar image" src="elliot.jpg">
  Elliot
</div>

23

Usage : replace the <div> element with an <a> element. 


<a class="ui label">

</a>

Tooltip

A label can point to content next to it

Please enter a value
Please enter a value
That name is taken!
Your password must be 6 characters or more

Usage :
- Add classes .tooltip, .below, .tooltip, .left, .tooltip, .right, .tooltip before the class .label for up, down, left and right point alternately.
- Add a class .inline to the field to display left or right tooltip.
- For showing background colors, using classes of colors.


<div class="field">
    <input type="text" placeholder="First name">
    <div class="ui tooltip label">Please enter a value</div>
</div>

<div class="field">
    <div class="ui below tooltip label">Please enter a value</div>
    <input type="text" placeholder="Last Name">
</div>

<div class="inline field">
    <input type="text" placeholder="Username">
    <div class="ui left tooltip  label">That name is taken!</div>
</div>

<div class="inline field">
    <div class="ui right tooltip label">Your password must be 6 characters or more</div>
    <input type="password" placeholder="Password">
</div>

Tag

A label can display as a tag which attached to someone or something for the purpose of identification or to give other information.

New Upcoming Featured

Usage : insert a class .tag before the class .label.


<a class="ui tag label">New</a>
<a class="ui error tag label">Upcoming</a>
<a class="ui info tag label">Featured</a>

Ribbon

A label can appear as a ribbon attaching itself to an element.

Overview Account Details Community User Reviews
Specs Reviews

Usage : insert a class .ribbon before the class .label. Add a class .right for displaying the ribbon at the right of box. 


<a class="ui primary ribbon label">Overview</a>
<a class="ui info ribbon label">Community</a>
<a class="ui warning right ribbon label">Specs</a>
<a class="ui error right label">Reviews</a>
