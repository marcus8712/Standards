Component

Button

Buttons provides our users a simple way to trigger an event, like searching for a query at a search engine, or to interact with dialog boxes, like confirming an action.

A standard button we use is a rounded rectangle with white background.

Usage : add classes .ui, .button to the <button> element. 


<button class="ui button">
    Submit
</button>

States

Disabled

A button shows its state which is currently unable to be interacted with.

Usage : insert a class .disabled before the class .button.


<button class="ui disabled button">
    Disabled
</button>

Active

A button shows its active state.

Usage : insert a class .active before the class .button.


<button class="ui active button">
    <i class="user icon"></i> Follow
</button>

Loading

A button shows a loading indicator.

Usage : insert a class .loading before the class .button.


<button class="ui loading button">
    Loading
</button>

Types

Icon

A button has only an icon.

Usage :
- Insert a class .icon to the class .button.
- Include an <i> element with classes .name, .icon. 


<button class="ui icon button">
  <i class="cloud icon"></i>
</div>

Animated

A button animates to show hidden content.

Next
Shop
Sign-up for a Pro account
$12.99 a month

Usage :
- Insert a class .animated before the class .button.
- Button is animated with a left-to-right direction by default. Insert classes .vertical or .fade to show vertical direction or fade effect.


<!-- Left to Right -->                         
<div class="ui animated button">
  <div class="visible content">Next</div>
  <div class="hidden content">
    <i class="right arrow icon"></i>
  </div>
</div>

<!-- Vertical -->                         
<div class="ui vertical animated button">
  <div class="hidden content">Shop</div>
  <div class="visible content">
    <i class="cart icon"></i>
  </div>
</div>

<!-- Fade -->                           
<div class="ui fade animated button">
  <div class="visible content">Sign-up for a Pro account</div>
  <div class="hidden content">
    $12.99 a month
  </div>
</div>

Basic Style

A basic button is less pronounced.

Usage : insert classes of color and a class .basic before the class .button.


<button class="ui basic button">Basic</button>
<button class="ui primary basic button">Primary</button>

Inverted

A button is formatted to appear on dark background.

Usage : insert classes of color and a class .inverted before the class .button.


<button class="ui inverted button">Standard</button>  
<button class="ui primary inverted button">Primary</button>

Variations

Rounded

A button is rounded with an icon.

Usage : insert a class .round before classes .icon.


<button class="ui round icon button">
  <i class="icon cloud download"></i>
</button>

Floated

A button is aligned to the left or right of its container.

Usage : insert classes .left, .floated or .right, .floated before the class .button.


<button class="ui right floated button">Right Floated</button>
<button class="ui left floated button">Left Floated</button>

Usage : insert a class .compact before the classes of .button.

Groups

Buttons

Usage :
- Add classes .ui, .buttons to the .<div> element.
- Include <button> elements with the classes .ui, .button. 


<div class="ui buttons">
  <button class="ui button">One</button>
  <button class="ui button">Two</button>
  <button class="ui button">Three</button>
</div>

Icon Buttons

Button groups show groups of icons.

Usage :
- Add classes .ui, .buttons to the <div> element.
- Include <button> elements with the classes .ui, .button.
- Add an icon with classes .name, .icon to the above elements, to show icon. 


<div class="ui icon buttons">
  <button class="ui button"><i class="bold icon"></i></button>
  <button class="ui button"><i class="font icon"></i></button>
  <button class="ui button"><i class="italic icon"></i></button>
</div>

Vertical Group

Groups is formatted to appear vertically.

Usage : insert a class .vertical before the class .buttons block.


<div class="ui vertical buttons">
  <button class="ui button">Feed</button>
  <button class="ui button">Messages</button>
  <button class="ui button">Events</button>
  <button class="ui button">Photos</button>
</div>
