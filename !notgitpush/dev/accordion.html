Plugin

Accordion

The accordion plugin makes it easy to make collapsing divisions of the page.

Initializing an accordion

The accordion method can be activated with a single line of JavaScript. 


<script>
$('.ui.accordion').accordion();
</script>

</div>

Usage : add an accordion block into any content block to add a new accordion.

    <div class="title">

    </div>

Accordion Menu

An accordion can be used to create content drawers inside a vertical menu.

Size
Colors

Usage :
- Insert a class .accordion before the classes .vertical, .menu.
- An item contains title and content.


<div class="ui accordion vertical accordion">
    <div class="item">
        <div class="title">...</div>
        <div class="content">...</div>
    </div>
</div>
