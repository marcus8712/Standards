Dropdown

A dropdown is a toggleable menu that allows the user to choose one value from a predefined list.

Initializing

Show the menu's hidden content via JavaScript : call an identifier of dropdown with a single line of JavaScript. 


<script>
$('.ui.dropdown').dropdown();
</script>

A Dropdown

Dropdown
Choice 1 Choice 2 Choice 3

Usage :
- Add classes .ui, .dropdown to the <div> element.
- Wrap menu in a .dropdown. Use <i class="dropdown icon"></i> to act as an indicator. 


<div class="ui dropdown">
  Dropdown <i class="dropdown icon"></i>
  <div class="menu">
    <a href="#" class="item">...</a>
  </div>
</div>

A Selection Dropdown

A selection is a type of dropdown that can be used to select between choices in a form.

Gender
Male
Female

Usage:
- Insert a class .selection before the class .dropdown.
- Include an <input> element with a class .hidden and a default text.


<div class="ui selection dropdown">
  <input type="hidden" name="gender">
  <i class="dropdown icon"></i>
  <div class="default text">Gender</div>
  <div class="menu">
    <div class="item" data-value="1">Male</div>
    <div class="item" data-value="0">Female</div>
  </div>
</div>

Selection can be initialized directly on a <select> element.


<select class="ui dropdown">
  <option value="">Gender</option>
  <option value="1">Male</option>
  <option value="0">Female</option>
</select>
