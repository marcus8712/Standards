Methods

Here are some useful methods that can be used with dropdown().

Method Description Example
Options Activates value of a dropdown. 
.dropdown({
    useLabels: false
})
Set text Sets dropdown text to a value. 
.dropdown('set text','text')
Refresh Refreshes all cached data. 
.dropdown('refresh')
Toggle Manually toggles a dropdown. 
.dropdown('toggle')
Show Manually opens a dropdown. 
.dropdown('show')
Hide Manually hides a dropdown. 
.dropdown('hide')
Clear Clears dropdown of selection. 
.dropdown('clear')

Options

There are certain options which can be passed via JavaScript to customize the look and feel of the Dropdown Window. Following table lists the options.

Option Name Default Value Description
1 Text Labels useLabels false Whether multiselect should use labels. Must be set to true when allowAdditions is true.
2 Maximum Selections maxSelections false When set to a number, sets the maximum number of selections.
3 User Additions allowAdditions false Whether search selection should allow users to add their own selections, works for single or multiselect.
4 Text Replacement action combo Changes text of previous sibling element.
5 Direction of Menu direction 'auto' When set to auto determines direction based on whether dropdown can fit on screen. Set to upward or downward to always force a direction.
6 Transitions transition auto (slide) Named transition to use when animating menu in and out. Defaults to slide down or slide up depending on dropdown direction. Fade and slide down are available without including ui transitions.
7 Duration duration 200 Duration of animation events.

Events

Following table lists the events to work with dropdown. These events may be used to hook into the function.

Event Description Example
onChange Is called after a dropdown value changes. Receives the name and value of selection and the active menu element. 
onChange: function (value, text, $choice) {
        // do something…                
    }
onAdd Is called after a dropdown selection is added using a multiple select dropdown, only receives the added value. 
onChange: function (addedValue, addedText, $addedChoice) {
        // do something…                
    }
onRemove Is called after a dropdown selection is removed using a multiple select dropdown, only receives the removed value. 
onChange: function (removedValue, removedText, $removedChoice) {
        // do something…                
    }
onShow Is called before a dropdown is shown. If false is returned, dropdown will not be shown. 
onShow: function () {
        // do something…                
    }
onHide Is called before a dropdown is hidden. If false is returned, dropdown will not be hidden. 
onHide: function () {
        // do something…                
    }
onNoResults Is called after a dropdown is searched with no matching values. 
onNoResults: function (searchValue) {
        // do something…                
    }
onLabelCreate Allows you to modify a label before it is added. Expects $label to be returned. 
onLabelCreate: function ($label) {
        // do something…                
    }
onLabelRemove Called when a label is remove, return false; will prevent the label from being removed. 
onLabelRemove: function (value) {
        // do something…                
    }
onLabelSelect Is called after a label is selected by a user. 
onLabelSelect: function ($selectedLabels) {
        // do something…                
    }
