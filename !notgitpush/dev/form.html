Component

Form

A form displays a set of related user input fields in a structured way.

Usage : add classes .ui, .form to the <form> element. 


<form class="ui form">
  ...
</form>

Contents

A Field Inside

A field is a form element containing a label and an input.

Usage :
- Add a <div> with a class .field into the form.
- Include a label and an input. 


<div class="ui form">
  <div class="field">
    <label>User Input</label>
    <input type="text">
  </div>
</div>
Inline Field

A field can have its label next to instead of above it.

Usage : insert a class .inline before the class .field. 


<div class="ui form">
  <div class="inline field">
    <label>Last name</label>
    <input type="text" placeholder="Full Name">
  </div>
</div>
Required

A field can show that input is mandatory.

Usage : insert a class .required before the class .field. 


<div class="ui form">
  <div class="required field">
    ...
  </div>
</div>

Fields

A set of fields can appear grouped together horizontally.

Usage : add many fields into a .fields. 


<div class="ui form">
  <div class="fields">
    <div class="field">...</div>
    <div class="field">...</div>
    <div class="field">...</div>
  </div>
</div>
Inline Fields

Multiple fields may be inline in a row.

Usage : insert a class .inline before the class .fields. 


<div class="ui form">
  <div class="inline fields">
    <label>Phone Number</label>
    <div class="field">...</div>
    <div class="field">...</div>
    <div class="field">...</div>
  </div>
</div>
Width

Fields can specify their widths in Grid Columns

Usage : insert classes of number and .wide before the class .field. 


<div class="ui form">
  <div class="fields">
    <div class="six wide field">...</div>
    <div class="four wide field">...</div>
    <div class="six wide field">...</div>
  </div>
</div>

Text Area Content

A textarea can be used to allow for extended user input.

Usage : add a <textarea> element after the label. To specify an approximate text area size use the rows attribute. 


<div class="ui form">
  <div class="field">
    <label>Text</label>
    <textarea></textarea>
  </div>
  
  <div class="field">
    <label>Short Text</label>
    <textarea rows="2"></textarea>
  </div>
</div>

  • Please enter your last name

<div class="ui form">
  <div class="ui message">
    <div class="header">We had some issues</div>
    <div class="ui bulleted list">
      ...
    </ul>
  </div>
</div>

States

Success

If a form is in an success state, it will automatically show any success message blocks.

You're all signed up for the newsletter.

Submit

Usage :
- Insert a class .success before the class .form.
- Add a success message into the form. 


<div class="ui success form">
  <div class="field">
    <label>E-mail</label>
    <input type="email" placeholder="marcus@hoiio.com">
  </div>
  <div class="ui success message">
    ...
  </div>
  <div class="ui button">Submit</div>
</div>

Variations

Equal Width

Forms can automatically divide fields to be equal width.

Usage : insert classes .equal, .width before the class .form. 


<div class="ui equal width form">
   <div class="fields">
      <div class="field">...</div>
      <div class="field">...</div>
   </div>
</div>

Fields can also automatically divide fields to be equal width.

Usage : insert classes .equal, .width before the class .form. 


<div class="ui form">
   <div class="equal width fields">
      <div class="field">...</div>
      <div class="field">...</div>
   </div>
</div>
