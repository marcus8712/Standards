Menu

A menu represents a group of commands that a user can perform or activate. This includes both list menus, which might appear across the top of a screen, as well as context menus, such as those that might appear underneath a button after it has been clicked.

Horizontal Menu

Editorials
Reviews
Upcoming Events

Usage :
- Add classes .ui, .menu to the <div> element.
- Include <div> element with a class.item.
- Insert a class .active or .disabled before the class .item to set item active or disabled.


<div class="ui menu">
    <div class="item">...</div>
    <div class="active item">...</div>
    <div class="disabled item">...</div>    
</div>

Contents of item

A menu varies according to the content of item.

Link Item

A menu can contain a link item, or an item formatted as if it is a link.

Visit Us
Contact Us

Usage : replace the <div> with an <a> element or insert a class .link before the class .item.


<div class="ui menu">
  <a href="http://www.hoiio.com" class="item">
    Visit Hoiio
  </a>
  <div class="link item">
    Contact Us
  </div>
</div>
Menu as an item

A menu can contain another menu group in the same level as menu items.

Browse Submit
Sign Up Help

Usage : Add a <div> element with a class .menu as an item into the menu block.


<div class="ui menu">
  <a class="item">...</a>
  <div class="menu">
     <a class="item">...</a>
  </div>
</div>
